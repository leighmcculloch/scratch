# scratch
A scratch repository. Ideas and attempts to build interesting things are in PRs. No merges.

If hacking on another repository in scratch:
- Clone the other repositories default branch into a new branch on this repository.
- Open a PR to track the work.
- Hack on the branch to build out the idea, experiment, or functionality.

If hacking on a new idea:
- Create a new branch for the idea.
- Open a PR to track the work.
- Hack on the branch to build out the idea.
